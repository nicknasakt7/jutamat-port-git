Team Flow Management System – Project & Task Management Platform

(Personal Project / Fullstack Developer)

FEATURED PROJECT

The application is designed to help small organizations manage their workflow more effectively, from project creation to task completion.

Users can create projects that consist of multiple tasks, where each task can be assigned to a specific employee with clear one-to-one ownership. Assigned users are responsible for updating task statuses ( In Progress Done ), while all members can collaborate through task-level comments.

The system includes role-based access control — Super Admin Admin Employee — allowing administrators to manage users, assign tasks, and oversee project progress.

To reduce manual tracking, the system automatically marks a project as completed once all tasks are finished, ensuring consistency across workflows.

A dashboard provides real-time visibility into overall performance, including completed tasks, pending work, overdue items, and progress tracking. Supports employee management with filtering by department and level.

Security is handled through password hashing, and users are able to manage their own credentials securely.

Architecture & Tech Stack

Next.js, NestJS, PostgreSQL, Prisma ORM, REST APIs, JWT (cookie-based), Cloudinary(file storage), TanStack React Query, NextAuth

Key Features

  • • Project and task management with multi-user assignment
  • • Role-based access control for SUPER_ADMIN, ADMIN, and MEMBER
  • • Task status tracking with real-time updates

  • • Designed relational database schema for projects, tasks, users, and role relationships
  • • Implemented task assignment logic supporting multiple users and role-based permissions
  • • Built backend services using NestJS, including API design, data flow, and business logic
  • • Developed frontend features using Next.js for project creation, task assignment, and status tracking

iSchool System – AI-driven Education Platform

(Project Leader / Fullstack Developer)

FEATURED PROJECT

iSchool is a school management platform that centralizes everything a school needs into one system — from student records and academic scoring to billing and parent communication.

Instead of juggling spreadsheets and paper forms, admins can manage students, teachers, and classrooms digitally. Teachers get a dedicated workspace to configure assessments, record scores, and track daily attendance. Parents stay informed through their own portal, where they can view their child's grades and pay school bills online via Stripe.

What sets iSchool apart is its AI-powered insight feature — powered by Google Gemini, the system automatically analyzes each student's performance and generates actionable feedback for both teachers and parents, helping schools identify at-risk students early and personalize learning support.

Built with a modern, production-ready stack — Next.js, NestJS, PostgreSQL, Prisma — designed to scale with the institution: multiple grades, classrooms, terms, and academic years out of the box.

Architecture & Tech Stack

Next.js, NestJS, PostgreSQL, Prisma ORM, REST APIs, JWT,Tailwind CSS,shadcn/ui, TanStack React Query, NextAuth(secure cookie-based authentication), Cloudinary (file storage), Google Gemini AI, Stripe, Gemini AI, Nodemailer (email)

Key Features

  • • Full student and teacher management with profile image uploads via Cloudinary
  • • Flexible per-term assessment configuration, score entry, and automatic GPA calculation
  • • Daily attendance tracking with school-wide summaries
  • • Stripe-integrated billing with bulk bill generation and online payments, including webhook confirmation
  • • Google Gemini AI-generated performance insights for students and classes with database caching
  • • Email-based parent invitation and onboarding flow using Nodemailer
  • • Student enrollment history tracking, including promotions, retentions, and transfers

Responsibilities & Contributions

  • • Led system design and defined technical workflows based on team discussions, ensuring shared understanding across frontend, backend, and data flow
  • • Developed core user flows (student, teacher, AI analysis) and backend services using NestJS
  • • Designed relational database schema and built promotion and class transition logic
  • • Implemented Stripe payment integration, TanStack Query state management, and Gemini AI integration
  • • Designed AI-based alert system and prepared project structure for team handover
  • • Handled edge cases including incomplete data, class transitions, and dynamic updates
